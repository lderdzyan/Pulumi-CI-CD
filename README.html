05 / Sprint Automation

🧠 GitHub Project Automation

An advanced script that queries GitHub Projects via GraphQL, tracks sprint state changes between days, and automatically comments on issues that lose their sprint assignments.

  1. Query GitHub ProjectV2
    Fetch all project items via GraphQL API using the GitHub CLI.
  2. Extract Sprint Fields
    Parse issue numbers and sprint field values using jq.
  3. Compare Snapshots
    Diff yesterday's state against today's to detect any sprint field changes.
  4. Detect Missing Assignments
    Flag issues that have lost their sprint assignment since the last run.
  5. Auto-Comment via GitHub CLI
    Post a warning comment on each affected issue automatically.
